## Logo Concepts

### Primary Logo Concept: "The Quirk"

**Design Direction:**

**Concept 1: Geometric Play**
- Wordmark "QUIRKABLE" with a distinctive quirk
- The letter "Q" features a 3D cube element emerging from it
- The cube appears to be mid-print, with visible layer lines
- Clean, modern sans-serif typeface
- Slightly rounded corners for approachability

**Visual Description:**
```
   ___
  |   |  QUIRKABLE
  |___|
 /     \
```

**Rationale:**
- The 3D cube immediately signals 3D printing
- The "quirk" in the Q represents problem-solving and creativity
- Layer lines suggest the printing process
- Modern and memorable

---

**Concept 2: The Print Line**

**Design Direction:**
- "QUIRKABLE" wordmark
- Horizontal print line running through the middle of all letters
- Line subtly steps up like 3D print layers
- Creates visual interest while suggesting the layer-by-layer process

**Visual Description:**
```
QUIRKABLE
‚îÅ‚îÅ‚îÅ‚îÅ‚îÅ‚îÅ‚îÅ‚îÅ‚îÅ  (stepped line showing layers)
```

**Rationale:**
- Subtle but distinctive
- Directly references 3D printing process
- Clean and professional
- Works at any size

---

**Concept 3: Modular Typography**

**Design Direction:**
- Custom letterforms built from geometric modules
- Each letter appears slightly "constructed" like 3D prints
- Maintains readability while showing craftsmanship
- Playful without being childish

**Rationale:**
- Unique and ownable
- Represents the build-it-yourself nature of 3D printing
- Memorable and distinctive
- Scales well

## Accessibility Guidelines

### Color Contrast

**WCAG AA Compliance (Minimum):**
- Normal text: 4.5:1 contrast ratio
- Large text (18pt+): 3:1 contrast ratio
- UI components: 3:1 contrast ratio

**Tested Combinations:**
‚úÖ Quirkable Blue on White: 8.2:1 (Excellent)
‚úÖ Quirkable Charcoal on White: 15.8:1 (Excellent)
‚úÖ White on Quirkable Blue: 8.2:1 (Excellent)
‚úÖ Precision Grey on White: 4.7:1 (Good)

**Never Use:**
‚ùå Creative Teal on White for body text (only 3.1:1)
‚ùå Energetic Coral on White for small text (only 3.8:1)

---

### Typography Accessibility

- Minimum body text: 16px
- Line height: 1.5-1.6 for body text
- Max line length: 75 characters
- Sufficient spacing between interactive elements (44px minimum touch target)

---

### Focus States

**All interactive elements must have visible focus states:**
- 2-3px outline in Quirkable Blue
- Offset by 2px from element
- Clear and obvious
- Never remove focus indicators

---

## Motion & Animation

**Philosophy:** Subtle, purposeful, never distracting

**Timing:**
- Fast: 150ms (microinteractions)
- Standard: 250ms (UI transitions)
- Slow: 400ms (page transitions)

**Easing:**
- Default: cubic-bezier(0.4, 0.0, 0.2, 1)
- Enter: cubic-bezier(0.0, 0.0, 0.2, 1)
- Exit: cubic-bezier(0.4, 0.0, 1, 1)

**Use Cases:**
- Button hover states
- Card lifting on hover
- Page transitions
- Loading states
- Micro-feedback animations

**Avoid:**
- Looping animations (except loading)
- Auto-playing videos with sound
- Parallax effects (can cause motion sickness)
- Excessive movement
