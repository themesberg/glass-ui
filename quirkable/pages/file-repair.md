## Section 2: Common File Problems We Fix

### Headline
**From Minor Glitches to Total Disasters**

### Subheadline
Here are the most common issues we encounter (and fix) every single day.

---

### Problem Card 1: Mesh Holes & Gaps

**Icon:** üï≥Ô∏è

**What It Is:**
Missing faces or edges in the 3D surface create "holes" where the model isn't closed. Printers can't figure out what's inside vs. outside.

**Symptoms:**
- Slicer shows errors
- "Non-watertight" warnings
- Model won't slice properly
- Random missing sections

**How We Fix It:**
We identify all gaps, close them intelligently (preserving design intent), and ensure a fully manifold, watertight mesh.

**Typical Fix Time:** 1-3 hours
**Cost:** ¬£12-30

---

### Problem Card 2: Non-Manifold Geometry

**Icon:** ‚ö†Ô∏è

**What It Is:**
Edges or vertices that don't properly define a solid volume. Multiple faces sharing one edge, internal faces, or impossible geometry.

**Symptoms:**
- "Non-manifold edges detected"
- Slicer confusion about inside/outside
- Model appears inside-out in places
- Unprintable despite looking okay

**How We Fix It:**
We analyze topology, separate problem geometry, rebuild connections properly, and ensure every edge is manifold.

**Typical Fix Time:** 2-4 hours
**Cost:** ¬£18-40

---

### Problem Card 3: Inverted Normals

**Icon:** üîÑ

**What It Is:**
Surface faces pointing the wrong direction‚Äîinside instead of outside or vice versa. Makes printers think solid is empty and empty is solid.

**Symptoms:**
- Model looks weird in slicer
- Wrong areas filled
- Inverted sections
- Bizarre slicing results

**How We Fix It:**
We flip all normals to point outward consistently, ensuring the entire model has correct orientation.

**Typical Fix Time:** 30 minutes - 2 hours
**Cost:** ¬£8-20

---

### Problem Card 4: Thin Walls & Unprintable Features

**Icon:** üìè

**What It Is:**
Sections too thin to print reliably with your chosen material/technology. Often from scaling or poor modeling.

**Symptoms:**
- Slicer warnings about thin features
- Model disappears when sliced
- Walls too thin to extrude
- Details lost

**How We Fix It:**
We identify thin sections, intelligently thicken walls while preserving design, or advise on alternative printing methods.

**Typical Fix Time:** 2-5 hours
**Cost:** ¬£20-50

---

### Problem Card 5: Intersecting Geometry

**Icon:** ‚öîÔ∏è

**What It Is:**
Multiple shapes overlapping or passing through each other, creating conflicting surfaces. Slicer doesn't know which surface to use.

**Symptoms:**
- Strange artifacts in sliced preview
- Random holes or fills
- Slicer confusion
- "Self-intersecting" errors

**How We Fix It:**
We separate intersecting parts, perform Boolean operations properly, and merge geometry cleanly.

**Typical Fix Time:** 2-4 hours
**Cost:** ¬£18-40

---

### Problem Card 6: Bad Scans & Photogrammetry Cleanup

**Icon:** üì∏

**What It Is:**
3D scans create noisy, artifact-filled meshes with missing data, noise, and rough surfaces.

**Symptoms:**
- Bumpy, noisy surface
- Missing chunks
- Scan artifacts and camera positions visible
- Unusable for printing

**How We Fix It:**
We smooth noise, fill missing sections, clean up artifacts, reduce poly count intelligently, and create print-ready geometry.

**Typical Fix Time:** 3-8 hours
**Cost:** ¬£30-80

---

### Problem Card 7: Unsupported Overhangs

**Icon:** üèîÔ∏è

**What It Is:**
Model design has angles too steep to print without support, or support would be impossible to remove.

**Symptoms:**
- Sagging or failed overhangs
- Impossible to support
- Design won't print as-is

**How We Fix It:**
We redesign problematic angles, add printable geometry, or reorient the model for optimal printing.

**Typical Fix Time:** 2-6 hours
**Cost:** ¬£20-60

---

### Problem Card 8: File Corruption & Format Issues

**Icon:** üíæ

**What It Is:**
Corrupted downloads, bad exports, incompatible formats, or damaged file data.

**Symptoms:**
- File won't open
- Import errors
- Partial model loading
- Strange geometry

**How We Fix It:**
We attempt recovery, convert between formats, rebuild from partial data, or work with you to get a better export.

**Typical Fix Time:** 1-4 hours
**Cost:** ¬£12-40

## Section 9: Prevention Tips (Free Advice)

### Headline
**How to Avoid File Errors in the Future**

### Subheadline
We're here to help, even if that means teaching you to avoid needing us!

---

### For CAD Users:

**‚úì** Export at reasonable resolution (not too high or too low)
**‚úì** Use "Save as Mesh" not "Export as Mesh" when possible
**‚úì** Check for non-manifold geometry before exporting
**‚úì** Verify units (mm, cm, inches) are correct
**‚úì** Use STL or 3MF formats for 3D printing

---

### For Downloaded Files:

**‚úì** Read comments on download sites for known issues
**‚úì** Check file in mesh viewer before sending to slicer
**‚úì** Download from reputable sources when possible
**‚úì** Keep original files as backup before modifications
**‚úì** Test small/fast prints before large expensive ones

---

### For 3D Scanners:

**‚úì** Capture more data than you think you need
**‚úì** Good lighting improves scan quality dramatically
**‚úì** Use scan cleanup tools immediately after scanning
**‚úì** Save "raw" scan data before any processing
**‚úì** Expect to need some repair‚Äîscans are never perfect
