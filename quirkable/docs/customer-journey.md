# Customer Journey

**How Quirkable Works - Step by Step**

---

## Overview

Quirkable's customer journey is designed to be seamless, transparent, and delightful from first contact to final delivery. Whether a customer is browsing ready-made models or starting a complex custom design project, the experience is guided, supportive, and professional.

---

## Journey Map by Service Type

### Journey 1: Buying Quirkable Originals

**Customer Type:** Gift buyer, hobbyist, collector

#### Stage 1: Discovery
**Touchpoint:** Website, social media, word-of-mouth, search

**Customer Actions:**
- Browses online catalogue
- Views product photos and descriptions
- Reads reviews and ratings
- Checks material and finish options

**Quirkable Provides:**
- High-quality product photography
- Detailed descriptions with dimensions
- Material and finish specifications
- Customer reviews and examples
- Clear pricing

**Emotions:** Curious, inspired, comparing options

---

#### Stage 2: Selection
**Touchpoint:** Product page

**Customer Actions:**
- Chooses model
- Selects material (PLA, resin, etc.)
- Picks finish (matte, gloss, painted)
- Customizes color (if applicable)
- Adds to cart

**Quirkable Provides:**
- Visual material/finish previews
- Price updates based on selections
- Estimated delivery time
- "Add to cart" confirmation

**Emotions:** Excited, confident in choice

---

#### Stage 3: Checkout
**Touchpoint:** Shopping cart and payment

**Customer Actions:**
- Reviews order
- Enters delivery information
- Selects shipping speed
- Completes payment

**Quirkable Provides:**
- Order summary with images
- Shipping options and costs
- Secure payment processing
- Order confirmation email

**Emotions:** Anticipation, trust

---

#### Stage 4: Production & Communication
**Touchpoint:** Email updates

**Customer Actions:**
- Waits for production
- Checks order status

**Quirkable Provides:**
- "Order received" confirmation
- "Printing started" notification
- "Quality check passed" update
- "Shipped" notification with tracking

**Emotions:** Anticipation, reassurance

---

#### Stage 5: Delivery
**Touchpoint:** Package arrival

**Customer Actions:**
- Receives package
- Unboxes print
- Inspects quality

**Quirkable Provides:**
- Secure packaging
- Branded presentation
- Care instructions
- Thank you note

**Emotions:** Delight, satisfaction

---

#### Stage 6: Post-Purchase
**Touchpoint:** Email follow-up, website

**Customer Actions:**
- Uses/displays the print
- Considers leaving a review
- Browses more products

**Quirkable Provides:**
- Review request email
- Discount code for next purchase
- Showcase of similar products
- Newsletter subscription option

**Emotions:** Satisfied, loyal, likely to return

---

### Journey 2: Bespoke Printing Service

**Customer Type:** Hobbyist, small business, DIY innovator

#### Stage 1: File Preparation
**Touchpoint:** Website upload form

**Customer Actions:**
- Has a 3D file (STL, OBJ, etc.)
- Navigates to "Upload for Printing"
- Drags and drops file

**Quirkable Provides:**
- Simple, intuitive upload interface
- Instant file size and format confirmation
- Automatic basic file check
- Tips for file preparation

**Emotions:** Hopeful, slightly uncertain

---

#### Stage 2: Review & Recommendations
**Touchpoint:** Automated analysis + manual review (if needed)

**Customer Actions:**
- Waits for printability assessment
- Reviews recommendations

**Quirkable Provides:**

- Material recommendations
- Size confirmation
- Suggested finish options

**Emotions:** Relieved (if file is good), grateful (if issues found)

---

**Customer Actions:**

- Picks quantity
- Adds rush service (optional)

- Transparent cost breakdown:

  - Print time
  - Post-processing
  - Delivery
- Total price calculator
- Estimated delivery date
- Options to modify and requote

**Emotions:** Confident, informed

---

#### Stage 4: Approval & Payment

**Customer Actions:**

**Quirkable Provides:**

**Emotions:** Excited, anticipation

### Journey 3: File Fixing & Repair

**Customer Type:** Anyone with a problematic 3D file

#### Stage 1: Problem Recognition
**Touchpoint:** External (failed prints, error messages)

**Customer Actions:**
- Tried to print file
- Got errors or failed prints
- Searched for solution
- Found Quirkable's repair service

**Quirkable Provides:**
- Educational content about common file issues
- "We can fix that" messaging
- Upload form for assessment

**Emotions:** Frustrated, hopeful

---

#### Stage 2: File Submission
**Touchpoint:** Repair service upload form

**Customer Actions:**
- Uploads problematic file
- Describes the issue
- Requests assessment

**Quirkable Provides:**
- Easy upload interface
- Form to describe the problem
- Explanation of what we check

**Emotions:** Hopeful, curious

---

#### Stage 3: Assessment & Quote
**Touchpoint:** Email with assessment results

**Customer Actions:**
- Receives detailed assessment
- Reviews recommended fixes
- Reviews pricing

**Quirkable Provides:**
- Detailed list of issues found:
  - Mesh holes
  - Non-manifold edges
  - Thin walls
  - Overhangs
- Fix recommendations
- Flat fee or hourly estimate
- Turnaround time
- Before/after preview (if possible)

**Emotions:** Informed, relieved there's a solution

---

#### Stage 4: Approval & Service
**Touchpoint:** Repair service execution

**Customer Actions:**
- Approves quote
- Pays for service
- Waits for repaired file

**Quirkable Provides:**
- Payment processing
- Progress updates (for complex repairs)
- Before/after comparison images
- Repaired file delivery
- Explanation of what was fixed

**Emotions:** Grateful, impressed

---

#### Stage 5: Delivery & Printing Option
**Touchpoint:** File download + print option

**Customer Actions:**
- Downloads repaired file
- Can choose to have Quirkable print it
- Or prints it themselves

**Quirkable Provides:**
- Clean, print-ready file
- Print recommendations
- Option to add printing service
- Discount offer if printing with Quirkable

**Emotions:** Satisfied, empowered

---

#### Stage 6: Feedback
**Touchpoint:** Follow-up email

**Customer Actions:**
- Uses the repaired file
- Shares success story

**Quirkable Provides:**
- "Did it print successfully?" check-in
- Request for testimonial
- Future project support offer

**Emotions:** Delighted, loyal

---

- **Every print inspected** before shipping

- **Branded packaging** with Quirkable logo
